# University Coursework

Repositorio con código de cursos, ayudantías y proyectos personales durante mi carrera en Ciencias de la Computación.

> ⚠️ Nota: Este repo se mantiene con fines de portafolio. Si eres estudiante actual de alguno de estos cursos, te recomiendo no copiar soluciones directamente — resuelve tus propias tareas.

## Estructura

| Carpeta | Contenido |
|---|---|
| `EDA/` | Estructuras de datos y algoritmos: hashing, backtracking, complejidad |
| `IIC2413-bases-de-datos/` | Concurrencia, recuperación, indexación, proyectos con PostgreSQL |
| `IIC2233-programacion-avanzada/` | Tareas, material de ayudantía, proyectos (juegos cliente-servidor, notebooks) |
| `IIC2713-ia-representacion-conocimiento/` | Proyectos de IA, PM Canvas, modelos de representación |
| `optimizacion-lineal/` | Simplex, teoría poliedral, MILP/MINLP |
| `asp-clingo/` | Programación por conjuntos de respuesta (Answer Set Programming) |
| `competitive-programming/` | Soluciones LeetCode en Python y C |

## Stack

Python · C · PostgreSQL · Clingo (ASP) · NumPy/Pandas · PyQt5
